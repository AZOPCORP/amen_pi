    ABOUT


    AMEN_PI IS AN AMEN BREAK SEQUENCER BASED ON THE DIGITS OF PI OR AN AUTOMATIC BREAKCORE GENERATOR.
    IT IS 100% CODED IN HTML5/JAVASCRIPT AND IT SHOULD RUN SMOOTH ON THE LATEST FIREFOX/CHROME VERSIONS.
    (CODED FOR FIREFOX BUT RUN BETTER ON CHROME (｀ε´) ...)
    IT IS BASED ON THE WEB AUDIO API AND USES JQUERY. / JQUERY KNOB FOR CONTROLS.
    THE CURRENT PI DIGITS IS SHOWN IS THE JS CANVAS
    THE MATRIX WORKS LIKE THIS: FOR N < 10 PI DIGITS YOU CAN ASSIGN ONE OF THE 10 AMEN CHOPPED SAMPLE
    THE FIRST POSITION IS ZERO SO THE RANGE IS 0 TO 9
    THE USER CAN GENERATE HIS OWN AMEN MATRIX OR LET IT BE GENERATED RANDOMLY
    IT SHOULD RUN REALTIME EXCEPT THE BITCRUSHER THAT NEED A FEW MILLISECONDS TO BUFFER (KEEP IT IN MIND WHILE RECORDING).
    WHEN THE PI SEQUENCE IS COMPLETED IT LOOP BACK TO THE FIRST ONE (GIVING AN UNDEFINED POSITION THAT I COULD NOT GET RID OFF, SORRY ...).
    THE FULL SOURCECODE IS AVAILABLE ON MYGITHUB IF YOU ARE CURIOUS.
